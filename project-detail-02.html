Project

Wine Brochure

+ Project Descriptions

Elizabeth is first group project and it makes with a 10 page website for a fictional winery. The website is built with HTML, Sass, and JavaScript and version-controlled with Git. It showcases simple custom logo, clean and user-friendly layout, and beautiful images to tell the winery’s story.

For this project, we started by creating wireframe with Aodbe XD. By using real images, colours, and fonts, the wireframe gave us a more realistic idea of what each page was going to look like after the development process. Also we built a prototype with Adobe XD. The interactive prototype was helpful for understanding the overall experience of the website even before it was built.
