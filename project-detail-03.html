React Movie Database

+ Project Descriptions

JM Flix is a movie app built using React and The Movie DB’s API. I have created a simple website where users can choose the kind of movies they want to watch and save their favorite movies.

The movie app needed to have four pages: home, about, individual movie, and favourites, and it needed to use the browser’s local storage to store users’ favourited movies to then be displayed on the favourites page. It also needed to be responsive and work on devices with a screen size of 320px to 1980px.
